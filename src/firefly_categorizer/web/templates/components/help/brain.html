Classification Engine

The Brain

When a transaction arrives, it goes through a three-stage decision process.

Memory Matcher

Checks for an exact match or a very close fuzzy match against your history. If you categorized "Netflix" before, it will recognize "Netflix.com".

Fastest

TF-IDF (Machine Learning)

If no exact match is found, it uses statistical analysis to compare the transaction description against all your past data to find similarities.

Smart

LLM (OpenAI)

As a last resort (and if configured), it asks an AI model to determine the category based on the description.

Fallback
