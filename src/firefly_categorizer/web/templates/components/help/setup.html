Configuration and Features

Tune automated behavior and connect real-time events.

Auto-Approve

You can set an AUTO_APPROVE_THRESHOLD (e.g., 0.9) in your config.yaml file. If the system is confident enough in its prediction, it will automatically update Firefly III without waiting for your confirmation.

Webhooks

The application can listen for real-time events from Firefly III. If you configure a webhook in Firefly to point here, new transactions can be categorized instantly as they are created.
